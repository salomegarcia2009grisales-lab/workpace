# Manual: Uso de GitHub desde Visual Studio Code 

## Paso 1: Iniciar sesión en GitHub desde VS Code
1. Abrir Visual Studio Code.
2. Abrir terminal
3. Seleccionar "Sign in with GitHub".
4. Autorizar el acceso desde el navegador.
5. Esperar a que VS Code confirme que la sesión fue iniciada.

## Paso 2: Abrir el proyecto
1. Ir a "File".
2. Seleccionar "Open Folder".
3. Elegir la carpeta del proyecto.
4. Verificar que los archivos aparezcan en el panel izquierdo.

## Paso 3: Crear o editar el archivo README.md
1. Buscar el archivo README.md en el explorador.
2. Abrirlo.
3. Escribir el contenido del manual en formato Markdown.
4. Guardar los cambios con Ctrl + S.

## Paso 4: Realizar el Commit desde VS Code
1. Hacer clic en el icono de Source Control (ramita).
2. Verificar que el archivo modificado aparezca en la lista.
3. Escribir un mensaje en el cuadro de texto (ejemplo: "Agrego manual en MD").
4. Presionar el botón de ✓ (Commit).

## Paso 5: Sincronizar los cambios
1. Después del commit, hacer clic en "Sync Changes".
2. Esperar a que los cambios se suban a GitHub.
3. Verificar en la página de GitHub que el archivo fue actualizado.
